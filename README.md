# Sovorn Chea Portfolio

Personal portfolio and research notebook for Sovorn Chea.

The site uses the [Chirpy][chirpy] Jekyll theme and is deployed with GitHub Pages.

## Focus

- AI agents and memory systems.
- Cognitive architectures.
- Causal inference and representation learning.
- Continual learning.
- Applied AI/ML research and engineering.

## Usage

Run through the GitHub Pages workflow or a local Jekyll environment with Ruby 3.1+.
